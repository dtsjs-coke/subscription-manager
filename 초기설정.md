파일이 총 12개예요. 순서대로 복사해서 만들어주시면 됩니다. 그리고 시작 전 꼭 해야 할 설정 3가지예요.
1. GitHub PAT 발급
github.com > Settings > Developer settings > Personal access tokens > Fine-grained tokens
→ Repository permissions에서 Contents: Read and write 설정
2. GitHub Actions Secrets 등록
저장소 Settings > Secrets and variables > Actions에 TELEGRAM_TOKEN, GH_PAT, GH_OWNER, GH_REPO 등록
3. Streamlit Cloud Secrets 등록
앱 대시보드 Settings > Secrets에 .streamlit/secrets.toml 내용 그대로 붙여넣기







🔐 2. GitHub Actions Secrets 등록
GitHub Actions가 실행될 때 텔레그램 토큰, GitHub PAT 같은 민감정보를 코드에 직접 쓰지 않고 안전하게 주입하는 방법이에요.
Step 1. 저장소 Settings 진입
본인 저장소 페이지 (github.com/유저명/subscription-manager)
→ 상단 탭 중 [Settings] 클릭
Step 2. Secrets 메뉴 찾기
왼쪽 사이드바
→ [Security] 섹션
→ [Secrets and variables]
→ [Actions] 클릭
Step 3. Secret 하나씩 등록
[New repository secret] 버튼 클릭 후 아래 4개를 각각 등록해요.
Name (이름, 정확히 입력)Secret (값)TELEGRAM_TOKENBotFather에서 받은 토큰 (예: 7123456789:AAF...)GH_PATGitHub PAT (아래 발급 방법 참고)GH_OWNER본인 GitHub 아이디 (예: john)GH_REPO저장소 이름 (예: subscription-manager)

⚠️ Name은 대소문자 구분 없이 저장되지만, notify.yml에 쓴 이름과 정확히 일치해야 해요.

[telegram]
token = "여기에_봇토큰_입력"

[github]
token = "여기에_PAT_입력"
owner = "여기에_GitHub아이디_입력"
repo = "subscription-manager"
```

→ `[Save]` 클릭 → 앱 자동 재시작

→ [Settings]
→ 왼쪽 맨 아래 [Developer settings]
→ [Personal access tokens]
→ [Fine-grained tokens]
→ [Generate new token]
입력 항목:
Token name     : subscription-manager-pat (자유롭게)
Expiration     : 원하는 만료 기간 선택 (1년 추천)
Repository access : Only select repositories
                    → subscription-manager 선택
Permissions    :
  Repository permissions
  └ Contents : Read and write ✅
→ [Generate token] 클릭 → 토큰 복사 (⚠️ 이 화면 벗어나면 다시 못 봐요!)
복사한 값을 GH_PAT Secret의 값으로 등록하면 돼요.

https://share.streamlit.io 접속
→ [Sign in with GitHub] 로그인
→ [Create app]
→ Repository: subscription-manager
→ Branch: main
→ Main file path: app.py
→ [Deploy!]
Step 2. App Settings 진입
배포된 앱 우측 하단 [⋮] 메뉴 또는
대시보드에서 앱 선택
→ [Settings]
→ [Secrets] 탭 클릭
Step 3. Secrets 내용 붙여넣기

[github]

owner = "여기에_GitHub아이디_입력"
repo = "subscription-manager"
```

→ `[Save]` 클릭 → 앱 자동 재시작

🤖 텔레그램 Bot 만들고 토큰 받는 방법
Step 1. BotFather 찾기
텔레그램 앱에서 검색창에 @BotFather 검색 → 파란 체크 있는 공식 계정 선택 → [시작] 또는 /start 입력
Step 2. 새 봇 만들기
/newbot 입력 후 전송
Step 3. 봇 이름 입력
BotFather가 물어봐요:
BotFather: "Alright, a new bot. How are we going to call it?"
나:  구독관리봇  (한글도 OK, 자유롭게)
Step 4. 봇 username 입력
BotFather: "Now let's choose a username for your bot.
            It must end in 'bot'."
나:  subscription_manager_bot  (영문, 반드시 bot으로 끝나야 함)

username은 전 세계에서 유일해야 해서 이미 있으면 다른 이름으로 바꿔야 해요.

Step 5. 토큰 받기
성공하면 BotFather가 이런 메시지를 보내줘요:
Done! Congratulations on your new bot. You will find it at t.me/subscription_manager_bot.
Use this token to access the HTTP API:

7123456789:AAFxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

Keep your token secure and store it safely.
이 긴 문자열 7123456789:AAFxxx... 이 바로 TELEGRAM_TOKEN 이에요. 복사해두세요!

📱 본인 Chat ID 확인하는 방법
봇 토큰과 별개로, 내 텔레그램 Chat ID 도 필요해요 (회원가입 시 입력하는 값).
텔레그램에서 @userinfobot 검색 → /start 전송하면 바로 알려줘요:
Your user ID: 987654321   ← 이게 Chat ID
이 숫자를 회원가입 페이지의 텔레그램 Chat ID 칸에 입력하면 돼요.

⚠️ 주의사항
봇을 만든 뒤 본인이 먼저 봇에게 말을 걸어야 메시지를 받을 수 있어요.
t.me/subscription_manager_bot 접속 → /start 전송 → 그 다음부터 알림 수신 가능!

여기까지 완료하면 TELEGRAM_TOKEN 값은 확보됐고, Chat ID는 회원가입 시 입력하면 돼요. 다음 단계 진행하다 막히면 알려주세요! 😊
